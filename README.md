## 서비스 소개
- PROFIT + FINANCE -> 이득이 되는 금융생활

- 다양한 목적으로 돈을 모으고 싶은 사람들을 위해, 빠르고 시간 낭비 없이 효율적인 금융 상품을 추천하기 위한 서비스

    1. 여러 은행의 예적금 서비스 정보를 수집, 실시간 제공하여 여러 금융 상품들의 정보를 한번에!
    2. 필터링 및 정렬 기능을 통해 일부 정보만 확인가능!
    3. 전체 사용자들의 가입 정보를 토대로한 기본/프리미엄 상품 추천 서비스!
    4. 꾸준히 이용하는 것만으로 서비스를 전부 이용 할 수 있는 출석체크 및 게시판 포인트 서비스!
    5. 그외 다양한 무료 서비스!

## 주요 기능

1. 금융 추천 알고리즘
    - 사용자들의 가입 인원 수를 토대로 한 기본 알고리즘
    - 사용자의 투자 성향 자산 등을 토대로 한 프리미엄 알고리즘(상세정보 제공)

2. 주변 은행 위치
    - 상품 가입 의사 결정 후 주변 은행 검색으로 어디로 가야할 지 간단하게 확인가능

3. 환율 계산기
    - 외국 여행을 목표로 돈을 모으는 사람들을 위해 목표 금액 설정을 위한 서비스

4. 커뮤니티/문의 게시판
    - 다양한 의견들을 나눌 수 있는 커뮤니티와 모르는 것들을 질문 할 수 있는 문의게시판

5. 예적금 상품 비교
    - 알고리즘 추천이 아닌 직접 하나하나 조건을 따지고 싶을때 필터링 기능을 포함한 예적금 상품 비교 툴

## 알고리즘

 - 일반 알고리즘
    - 일반 알고리즘의 목표는 '사람들의 선호도' 이다.
    - 가입한 사람들을 가중치로, 상품을 정렬하는 간단한 알고리즘
    - 20대가 원하는 목표, 40대가 원하는 목표가 다를 수 있어 연령별 가중치를 다르게 설정

 - 프리미엄 알고리즘
    - 초기 설정하였던 투자 성향을 통한 분류
    - 목표/보유 자금을 통해 1차적인 필터링 진행
    - 안전형/수익형/무계획 3가지 분류로 진행

    - 저축 기간(단기,장기)/이자율/ 변동성 or 적립 방식 을 기반으로 기초 '상품 옵션별' 기준 가중치를 설정
    - 이때 무계획은 안전형과 수익형의 평균으로 전체적인 가중치 계산은 안전형의 영향을 높이 받게 설정

    - 기준 가중치에 인원수 가중치를 곱해줌, 이때 안전형/수익형의 경우 인원 가중치의 영향 많이X
